# 穿越到异世界，挑选你的队友

这是一个伪装成“异世界队友测评”的病娇视觉小说问卷。表面流程是为玩家生成冒险小队，实际流程是“第 0 名队友”逐步接管题目、篡改记录，并把玩家引向不同结局。

公开页面：

```text
https://littleornage.github.io/
```

## 本地预览

```powershell
node preview-server.mjs
```

然后访问：

```text
http://127.0.0.1:5173/
```

如果 Codex 内置浏览器提示 `ERR_BLOCKED_BY_CLIENT`，这是当前内置浏览器客户端的本地地址拦截策略，不是网页文件错误。此时用系统浏览器打开上面的地址即可预览。

## 配置入口

- `story-config.js`：30 道题、选项、隐藏数值、系统日志、结局文案。
- `app.js`：问卷流程、系统日志、选项变字、最终分支逻辑。
- `styles.css`：问卷界面、故障选项、系统日志、弹窗、结果报告样式。

## 隐藏数值

页面会记录前排、魔法、治疗、探索、策略、羁绊等表层倾向，也会记录依赖、警觉、支配、嫉妒、真相、轮回、逃离等隐藏倾向。第 30 题提交后会根据这些数值进入不同结局。
