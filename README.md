# 视觉小说问卷页

这是参考页的本地可配置还原版。直接打开 `index.html` 即可运行；页面逻辑读取 `story-config.js`，后续改题目、选项、异常文字和日记页时优先改配置文件。

## 本地预览

推荐在项目目录运行：

```powershell
node preview-server.mjs
```

然后访问：

```text
http://127.0.0.1:5173/
```

如果 Codex 内置浏览器提示 `ERR_BLOCKED_BY_CLIENT`，这是当前内置浏览器客户端的本地地址拦截策略，不是网页文件错误。此时用系统浏览器打开上面的地址即可预览。

## 配置入口

- `title`：问卷标题。
- `startSlides` / `warning`：进入问卷前的弹窗文案。
- `questions`：30 道题和每题 4 个选项。
- `effects`：中途异常事件文案。
- `final`：提交、解锁、结尾流程文案。
- `diary`：最终留言页，每页可以写 `date` 和 `html`；最后一页使用 `final: true`。

## 已还原的特殊节点

- 第 3 题：选项后黑屏打字，并自动跳到下一题。
- 第 5 题：从第 4 题进入后，选项短暂被替换。
- 第 8 题：答题后红屏、漂浮文字，并自动跳题。
- 第 16 题：弹出小型红框提示。
- 第 17 题：部分选项被强制锁定，部分选项进入全屏打字。
- 第 20 题：选项文字被篡改，前三个被锁定。
- 第 25 题：全屏窥视文字覆盖，选项短暂失效。
- 提交后：强制查看留言、输入指定文字、日记翻页和最终确认。
